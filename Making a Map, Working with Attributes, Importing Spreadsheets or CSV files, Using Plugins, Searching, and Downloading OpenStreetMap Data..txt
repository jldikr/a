AIM: Making a Map, Working with Attributes, Importing Spreadsheets or CSV files, Using Plugins, Searching, and Downloading OpenStreetMap Data.

Part 1 - India Map:
1. Layer → Add Vector → Select image (adm0_shp).
2. Project → New Print Composer → Give name.
3. A new interface opens.
4. Layout → Add map → Select and drag.
5. Layout → Add image → Select and drag → Go to item → Search directories → Select image.
6. Layout → Add label → Select and drag → Write any text.
7. Layout → Add scale bar → Select and drag.
8. Layout → Add arrow → Select and drag.
9. Layout → Add legend → Select and drag.
10. Layout → Add attribute/table → Select and drag.
11. Left side select shapes → Select and drag → Go to item → Change color.
12. Composer → Export as Image → Give Name → Save.

Part 3 - Sample File:
1. Select Layer → Add Layer → Add Delimited Text Layer.
2. File Name → Select sample xls.
3. Select custom delimiters → Select all checkboxes.
4. In geometry definition → X field (select LATITUDE) → Y field (select LONGITUDE).
5. Click OK → Close → Cancel.
